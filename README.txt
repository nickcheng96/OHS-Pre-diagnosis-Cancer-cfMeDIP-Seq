The code here is used in the study 'Pre-diagnosis plasma cell-free DNA reveals early signatures of prostate and breast cancer risk up to eight years prior to clinical detection' by Cheng et al. (2025).

The bash ".sh" files contain the order to run R scripts for performing count merging and normalization, differential methylation analysis, characterization of DMRs across genomic elements, genomic tissue methylation comparison and predictive modelling. 

All files needed to run code is available in github aside from OHS raw methylation count files.

OHS cfMeDIP-Seq methylation raw covergae count data for 300bp bins, silencer and enhancer regions can be found here:
https://zenodo.org/records/18041305
